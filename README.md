# RD Assistente

Sistema inteligente de gestão para a RD Pizzaria.

## Objetivo

Centralizar o controle financeiro da pizzaria através do WhatsApp utilizando Inteligência Artificial.

## Funcionalidades previstas

- Cadastro de compras
- Separação Loja x Pessoal
- Categorias automáticas
- Dashboard financeiro
- Controle de estoque
- Relatórios
- Integração com WhatsApp
- IA para interpretar mensagens
- OCR para leitura de notas fiscais

## Tecnologias

- Node.js
- Express
- Supabase
- OpenAI
- Evolution API
- GitHub
